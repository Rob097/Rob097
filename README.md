# Hey, I'm Roberto

Full stack developer from Italy with about five years of professional experience. My day-to-day revolves around React and Spring Boot, though I've worked across most of the stack at some point.

At Fleap I build and maintain production systems in the fintech space. Most of what ships there runs on Kubernetes and follows a microservices architecture. Before that I spent two years at GPI Group working on healthcare management software.

I keep side projects going. Some are experiments, some grew into something real. A few of them are below.

## Projects

**[EdilSync](https://github.com/Rob097/edilsync-experimental)** — Construction site coordination platform. Keeps everyone aligned on a residential project: contractor, client, subcontractors and professionals all in one place, with tasks, appointments, documents and decisions always tracked. Built with React, Vite, Tailwind and Supabase. Live at [edilsync.rdlabs.digital](https://edilsync.rdlabs.digital/).

**[MyPortfolio](https://github.com/Rob097/MyPortfolio-FE-React)** — Platform for building detailed professional portfolios, organized as stories rather than plain CV entries. Microservices backend with Spring Boot, React microfrontend architecture on the frontend.

**[Spring React Multitenant Boilerplate](https://github.com/Rob097/spring-react-multimodule-multitenant)** — Starting point for multi-tenant web apps. Spring Boot multimodule backend with a separate MongoDB database per tenant, React frontend.

**[Spring Boot Enterprise Starter](https://github.com/Rob097/spring-boot-enterprise-starter)** — Opinionated starter for Spring Boot enterprise applications with multi-module structure, Spring Security and JWT authentication.
